# 2026-03-20 工作日志

## LLM2Rec 数据预处理 Notebook 创建完成

为用户创建了 3 个 Jupyter Notebook，实现从原始数据到 LLM2Rec 全部训练/评估文件的完整预处理流水线：

### 创建的文件
1. `notebooks/01_download_raw_data.ipynb` — 从 Amazon Reviews 2023 (8领域) 和 Goodreads 官方源下载原始数据，流式解析 .jsonl.gz，保存为 Parquet 中间文件
2. `notebooks/02_filter_and_split.ipynb` — 迭代式 5-core 过滤、ID重编号(从1开始)、按时间排序构建用户序列(max_len=10)、leave-one-out划分、生成评估文件(data.txt/train_data.txt/val_data.txt/test_data.txt/item_titles.json)、与论文统计对照验证
3. `notebooks/03_generate_training_data.ipynb` — 生成 AmazonMix-6 CSFT CSV、item_titles.txt、training_item_pairs_gap24.jsonl、单领域训练CSV、完整性校验
4. `requirements_data.txt` — 数据预处理所需的 Python 依赖

### 数据集范围
- 训练 (AmazonMix-6): Video_Games, Arts_Crafts_and_Sewing, Movies_and_TV, Home_and_Kitchen, Electronics, Tools_and_Home_Improvement
- OOD评估: Sports_and_Outdoors, Baby_Products, Goodreads

### 关键数据格式约定
- item ID 从 1 开始（0 保留为 padding）
- item_titles.json: key从"1"开始，不含"0"
- CSFT CSV: history_item_title 和 history_item_id 是 Python list 的 str(repr) 形式，用 eval() 解析
- training_item_pairs_gap24.jsonl: 整体是一个 JSON 数组（虽然后缀是 .jsonl）
